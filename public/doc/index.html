REST API

Все запросы методом POST
Формат запроса: 

                                
                                    {"id":0,"params":{"offset":0,"limit":100}}

если нет тпараметров, свойство params = {}

Для запросов требующих авторизацию передаём header с токеном полученным при авторизации 

                                
                                    Authorization: токен_пользователя

Simple Sidebar

This template has a responsive menu toggling system. The menu will appear collapsed on smaller screens, and will appear non-collapsed on larger screens. When toggled using the button below, the menu will appear/disappear. On small screens, the page content will be pushed off canvas.

Make sure to keep all page content within the #page-content-wrapper.
